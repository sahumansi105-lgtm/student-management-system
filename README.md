# 🎓 Student Management System

## 📌 Overview
A React-based Student Management System with full CRUD operations, search, and pagination.

---

## ⚙️ Features
- Add Student
- Edit Student
- Delete Student
- Search Students
- Pagination
- Toast Notifications
- Responsive UI

---

## 🛠️ Tech Stack
- React.js
- Axios
- CSS
- Node.js + Express (Backend)
- Prisma ORM

---
