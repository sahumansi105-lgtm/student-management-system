# 🎓 Student Management System

## 📌 Overview
A full-stack Student Management System built using React.js for frontend and Node.js + Express for backend.  
It performs full CRUD operations with search, pagination, and toast notifications for better user experience.

---

## ⚙️ Features
- ➕ Add Student
- ✏️ Edit Student
- ❌ Delete Student
- 🔍 Search Students
- 📄 Pagination
- 🔔 Toast Notifications
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Axios
- CSS (Custom Styling)

### Backend:
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL / MySQL
